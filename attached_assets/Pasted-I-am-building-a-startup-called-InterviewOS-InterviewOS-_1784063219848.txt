I am building a startup called InterviewOS.

InterviewOS is an AI-powered interview operating system that helps software engineers prepare for coding, system design, machine learning, behavioral, and company-specific interviews.

This is NOT a student project.
The UI should feel like a polished Silicon Valley startup (OpenAI, Linear, Notion, Vercel, Arc Browser, Stripe Dashboard, Anthropic) while still retaining Google's Material Design philosophy.

The design language should follow Stock Android / Pixel UI.
